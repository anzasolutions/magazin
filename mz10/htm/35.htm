/fhelp status:
+-----------------------------------------------------+
| _ 555ex ver. 1.3 - Symbole linii statusowej_ |
+-----------------------------------------------------+
| Symbole rozszezonej linii statusu [0-0:GSCNXALFMPHRX] oznaczaja: |
| A = Friends CTCP HELP/OP/INVITE/UNBAN/CHOPS L = Logowanie IRC |
| M = Protekcja Mass Deop/Kick G = XDCC AutoGet |
| F = Zabezpieczenia anty Flood'owe N = XDCC AutoRename |
| H = Zabezpieczenia Netsplit Hack/Ban C = XDCC AutoClose |
| P = Zabezpieczenia Flood'ow na kanal/do Nicka S = XDCC AutoSend |
| R = Automatyczne wlazenie na kanal po kopie X = XDCC Security |
| 0-0 = Pliki oferowane - pliki czekajace na odbior. |
| \(Duza litera == opcja wlaczona , mala == opcja wylaczona\) |
